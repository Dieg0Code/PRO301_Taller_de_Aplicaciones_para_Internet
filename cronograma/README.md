# Cronograma Módulo PRO301 Taller de Aplicaciones para Internet

## Información General

- Período: 16 de Marzo - 20 de Mayo de 2026
- Horario: Lunes, Martes y Miércoles de 10:50 a 13:10
- Total: 90 horas

## Enfoque del Módulo

El módulo se organiza con foco en:

- fundamentos transferibles de arquitectura web, frontend, datos y aplicaciones para Internet;
- interfaces accesibles, responsivas y mantenibles;
- integración entre componentes, servicios y persistencia;
- trabajo con herramientas reales de desarrollo, inspección y validación técnica;
- uso responsable de agentes e IA como apoyo de análisis, generación, depuración y documentación;
- y comprensión aplicada de tecnologías emergentes relevantes para el desarrollo actual.

## Resultado Esperado del Módulo

Al finalizar, el estudiante debería ser capaz de:

- Comprender la arquitectura de una aplicación web moderna más allá de un lenguaje específico.
- Construir interfaces accesibles y responsivas con herramientas actuales.
- Integrar frontend, APIs y bases de datos con criterios de mantenibilidad.
- Leer y adaptar código legado cuando el contexto institucional o laboral lo requiera.
- Utilizar agentes de IA como apoyo de desarrollo sin delegar el juicio técnico.
- Entender los fundamentos de deep learning y su relación con productos basados en LLMs.
